# 🎬 核心实战：TMDB 影史全维度分析

这是本项目最具挑战性的部分，涵盖了从原始 JSON 解析到高阶统计可视化的全流程。

## 1. 技术栈亮点

- **数据清洗**：使用 `jsonlite` 处理嵌套字段，利用 `tidyr::separate_rows` 实现流派的一对多拆解。
- **统计建模**：对 Budget 与 Revenue 进行 **双对数空间相关性分析**，发现变温规律。
- **时间序列**：利用 `lubridate` 揭示了电影行业的“季节性规律”与“题材代际演变”。
- **文本挖掘**：通过 `tidytext` 提取了不同流派的核心关键词画像。

## 2. 核心洞察

- **财富密码**：恐怖片（Horror）以极低的均摊成本获得了最高的中位数 ROI，是商业上的避风港。
- **质量标杆**：纪录片（Documentary）在评分维度上表现最稳健且均值最高。
- **档期效应**：全球电影票房呈现明显的双峰效应，6月与12月是绝对的黄金档期。
