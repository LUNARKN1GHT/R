# R 编程学习仓库

本仓库用于托管 R 语言学习过程中的代码。内容以《R语言核心技术手册》这本课本为基础，同时让 Gemini 老师进行辅助。仓库中存储基础代码文件和数据文件。

欢迎交流！

## 与 python 的对比

> 快速掌握一门计算机语言，我们不要从语言特性出发，而应该从**语言共性**出发。

R 语言和 python 是比较相似的，内容简洁、生态丰富且功能强大。下面这个表格给出 python 和 R 语言的不同之处。

|特性|Python|R|
|:---:|:---|:---|
|核心定位|通用变成、生产环境部署|统计分析、科学研究、数据可视化|
|数据处理|主要为 `pandas` 库|原生支持 `DataFrame`|
|绘图方式|主要为 `Matplotlib`库|`ggplot2`（基于图形语法层级叠加）|
|语法习惯|0索引，用 `=` 赋值|1 索引，习惯用 `<-` 赋值|

## 仓库结构

- `/programs`: 存放课本的相关内容与笔记。文件夹按章节划分为 `chapter_{index}`，文件夹下存放相应章节的各个知识点的代码和笔记。
- `/data`: 存放仓库中各个程序使用的数据集。
- `/projects`: 存放用 R 编写的实战项目。项目处理结果与相应说明文档均存放在相应文件夹下。
  - `/projects/heart_disease`: 心脏病数据分析，是一个经典的数据分析与机器学习数据集。
- `/install.R`: 包安装程序。设置清华镜像站点安装。

## 一些说明

### 关于 R 语言

- 相比大家都不会喜欢 R 语言那个~~老的掉牙的图形化界面~~，没有格式化没有提示补全写起来相当便秘。所以最好还是在 VS Code 上进行编程。
- R 语言的版本有很多，本仓库使用的版本是 4.4.0，R 社区生态关于向下兼容疑似做得不太好（至少 `nutshell` 这个包我始终用不了）。**迁移并运行代码的时候务必注意版本号**！

### 关于《R语言核心技术手册》

- 这本书更像一本**工具书**，而非课堂上使用的教材。这本书只能提供编程上的一些参考，有需要的时候再翻查相关知识点。
- 正如之前提到的，我们编程的时候首先要从语言之间的共性出发，我们总不可能重新再学一遍 `a + b` 之类的简单代码。所以书中很多内容仓库都没有进行复现。
- 书中使用的 R 版本似乎比较古早，有些特性在现在不一定能使用。这里注意辨别。
- 书中反复使用到 `nutshell` 这个包，但是这个包过于古早，不符合本仓库的 R 版本。我索性就放弃了书中一些示例。

### 关于编程

- ==程序是算法思想的具体实现，算法是程序的灵魂==，学习编程的时候，更应该关注为了做什么事情写这条语句，而不是机械地背某个函数的用处（应用的时候翻翻书就好）。
- 看书也不能感受到这门语言的使用情景与语言特性，所以，想要学好一门语言，还是要在一个情境下进行分析。本仓库的 `/projects` 正是基于这个目标出发的。
- 在AI时代，编程还需要人手敲吗。让 gpt 老师或 Gemini 老师帮你写才是版本答案！
